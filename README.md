# Islam Salahuddin's Portfolio Website

A sophisticated, multi-personal portfolio website showcasing the diverse professional facets of Islam Salahuddin. This interactive portfolio presents six distinct professional personas, each with their own dedicated section, projects, and status tracking.

## Live Demo

**[View Live Portfolio](https://islamsalahuddin.github.io/)**

## Overview

This portfolio website is designed as an interactive experience where visitors can explore different sides of Islam's professional identity:

- **The Ex-Journalist** - Retired journalism background and experience
- **The Technical Writer** - Paused technical writing capabilities  
- **The Creative Writer** - Ongoing creative writing projects
- **The 'Data Guy'** - Current data analytics and BI expertise
- **The MSc AI Student** - Active academic journey in Artificial Intelligence
- **The Analytics Facilitator** - Paused facilitation and training role

Each persona is represented by a custom character illustration and includes detailed professional information, project showcases, and current status indicators.

## Design & Features

### Visual Design
- **Color Scheme**: Warm, inviting palette with coral accents (#b97552) and light backgrounds (#FBF7F2)
- **Typography**: Clean, modern fonts (Segoe UI, Poppins, Open Sans) for optimal readability
- **Layout**: Responsive grid system with mobile-first approach
- **Interactive Elements**: Custom carousels, hover effects, and smooth transitions

### Key Features
- **Interactive Persona Carousel**: 3D carousel showcasing all six professional personas
- **Status Tracking**: Real-time status indicators (Ongoing, Paused, Retired) for each persona
- **Project Showcases**: Comprehensive project galleries with role attribution
- **Professional Updates**: Embedded LinkedIn feed for latest professional activities
- **Skills Visualization**: Interactive skill chips with hover effects
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices

## Technology Stack

### Frontend Technologies
- **HTML5**: Semantic markup with accessibility considerations
- **CSS3**: Advanced styling with animations, transitions, and responsive design
- **JavaScript (ES6+)**: Interactive features, carousel functionality, and dynamic content
- **Bootstrap 4.6.0**: Responsive grid system and UI components

### External Dependencies
- **jQuery 3.6.0**: DOM manipulation and event handling
- **Bootstrap JS**: Carousel and interactive components
- **Google Fonts**: Poppins and Open Sans typography
- **LinkedIn Embed API**: Professional updates integration

### Build Tools
- **No build process required**: Static site ready for direct deployment
- **GitHub Pages**: Hosting platform with automatic deployment from main branch

## Responsive Design

The website is fully responsive with breakpoints at:
- **Desktop**: 1200px+ (Full carousel experience)
- **Tablet**: 768px-1199px (Adapted layouts)  
- **Mobile**: 576px-767px (Simplified navigation)
- **Small Mobile**: <576px (Optimized touch interactions)

## Deployment

This repository is configured for GitHub Pages deployment:

1. **Source**: Deploy from the `main` branch
2. **Root**: `/` directory (custom domain ready)
3. **HTTPS**: Enabled by default
4. **CDN**: GitHub's global CDN for fast loading

### Local Development
Simply clone the repository and open `index.html` in a web browser - no build process required.
